## Обзор

В этом репозитории содержится исходный код «VR-тренажёра публичных выступлений» (VR Public Speaking Trainer, сокращённо ТПВ) — приложения виртуальной реальности, предназначенного для моделирования сценариев публичных выступлений с целью тренировки ораторских навыков и преодоления страха перед сценой. Проект разработан в качестве курсовой работы по дисциплине «Технология разработки программных систем» в Московском государственном техническом университете имени Н. Э. Баумана (МГТУ им. Н. Э. Баумана), факультет информационных технологий и систем управления, кафедра ИУ6 «Компьютерные системы и сети», направление 09.03.01 «Информатика и вычислительная техника».

Приложение моделирует реалистичные условия выступления, в которых пользователи могут репетировать речи перед виртуальной аудиторией, настраивать реакцию аудитории, фиксировать и анализировать речевые показатели (продолжительность и среднюю громкость), а также создавать отчёты о сессиях. Он предназначен для образовательных учреждений, корпоративного обучения и личного использования.

 Разработка основана на Unity 2022.3 LTS с OpenXR для кроссплатформенной совместимости с виртуальной реальностью, что обеспечивает поддержку таких устройств, как HTC Vive Pro Eye и других, соответствующих стандартам OpenXR (в данный момент поддерживается только HTC Vive Pro Eye, т.к. проект находится в стадии прототипа).

---
## Основные функции

- **Запуск симуляции**: пользователи выбирают один из трёх сценариев (конференц-зал, лекторий, актовый зал) и запускают сеанс виртуальной реальности на виртуальной сцене, где их окружают 10–120 неигровых персонажей (NPC) с заранее заданными анимациями поведения.
- **Управление реакцией аудитории**: во время сеанса с помощью контроллера можно переключать режимы реакции аудитории (положительная, нейтральная, отрицательная), при этом поведение и звуки NPC будут мгновенно реагировать (<1 секунды) (например, аплодисменты при положительной реакции, зевота или шёпот при отрицательной).
- **Запись и анализ речи**: записывает речь пользователя через встроенный микрофон VR-устройства, измеряет продолжительность сеанса в секундах и среднюю громкость в децибелах (дБ). Сеансы можно прерывать или ставить на паузу с помощью команд контроллера.
- **Создание отчета о сеансе**: в конце сеанса отображается отчет, включающий информацию о продолжительности, средней громкости речи, количестве переключений режима реакции и общем времени сеанса. Отчеты сохраняются локально в формате JSON.
- **Настройка параметров**: предварительные настройки для аудитории (10–120 NPC), ожидаемая продолжительность и уровень сложности (базовый: пассивная аудитория; продвинутый: случайные события, например вопросы из зала).
- **Функции обеспечения надёжности**: проверка ввода (например, количество человек должно быть целым числом от 10 до 120), блокировка недопустимых действий (например, выход за пределы сцены приводит к телепортации обратно) и целостность данных за счёт хеширования локальных файлов отчётов в формате JSON.
- **Встроенная справочная система**: интегрированная справочная система, доступная через меню виртуальной реальности, с инструкциями по использованию.
- **Оптимизация**: обеспечивает частоту кадров не менее 90 на минимальном оборудовании (Intel Core i5-8400/AMD Ryzen 5 2600, 8 ГБ ОЗУ, NVIDIA GTX 1060/AMD RX 580, SSD-накопитель на 10 ГБ). Использует отбраковку по окклюзии, статическую пакетную обработку, уровни детализации для неигровых персонажей и эффективные шейдеры.
- **Распространение**: сборка в отдельный исполняемый файл VRPublicSpeakingTrainer.exe, включающий все ресурсы Unity и библиотеки OpenXR, не требующие дополнительных зависимостей, кроме драйверов VR-устройств.

## Технические требования

- Процессор: Intel Core i5-8400 или AMD Ryzen 5 2600.
- Оперативная память: 8 ГБ.
- Видеокарта: NVIDIA GeForce GTX 1060 или AMD Radeon RX 580.
- Хранилище: 10 ГБ на SSD.
- Устройство виртуальной реальности: совместимое с OpenXR (например, HTC Vive Pro Eye) с микрофоном и контроллерами (минимальное разрешение 1440x1600 на каждый глаз, частота обновления 90 Гц).

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/NazarenkoYura/VRPublicSpeakingTrainer
   ```

2. Откройте проект в Unity Hub (версия 2022.3 LTS или более поздняя).
   - Убедитесь, что плагины OpenXR, XR Plugin Management и XR Interaction Toolkit установлены через менеджер пакетов.

3. Соберите автономную версию для Windows:
   - Выберите «Файл» > «Настройки сборки».
   - Выберите платформу Windows.
   - Соберите файл .exe.

4. Установите драйверы для VR-устройств (например, SteamVR для HTC Vive) и убедитесь, что установлена среда выполнения OpenXR.

 Внешние базы данных или серверы не требуются; все операции выполняются локально.

---
##  Использование

1. Запустите файл .exe на совместимом ПК с подключенной VR-гарнитурой.
2. В главном меню (собственный VR-интерфейс):
   - Выберите сценарий.
   - Установите количество человек в аудитории (10–120).
   - Установите начальную реакцию и целевую продолжительность.
   - Начните занятие.
3. Во время занятия:
   - Говорите в микрофон.
   - Переключайте режимы аудитории с помощью кнопки на контроллере.
   - Приостанавливайте и возобновляйте занятие с помощью интерфейса на запястье.
4. Завершите занятие вручную или по таймеру; просмотрите и сохраните отчет.
